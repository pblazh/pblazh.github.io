SANTA CLIENT

интерактивная новогодняя открытка

Клиент: портал о креативе в рекламе Adme.UA, Агентство: Peppermint interactive

интерактивная новогодняя открытка

http://santaclient.com/

Задача

Разработать новогоднее поздравление для посетителей рекламного портала ADME.UA.

Реализация

В Новый год все ждут Деда Мороза с подарками, а рекламисты ждут подтверждения бюджетов.

Был создан виртуальный митинг-рум Santa Clientа, где каждый посетитель может презентовать свое желание и побороться за бюджет на его исполнение. Для каждого запроса пользователя выбирался максимально соответствующий видео-сюжет, а из поиска изображений google изображение, которое бы выигрышно смотрелось при наложении на видео ролик.

Во время запуска сайта мы столкнулись с проблемой. Сайт пользовался такой популярностью, что мы моментально выходили за лимит бесплатного пользования API Google. Пришлось срочно реализовать кэширование для поиска изображений.

Проект выполнен на технологии Flash.
