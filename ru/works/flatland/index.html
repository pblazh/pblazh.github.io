Плоский Мир

https://pblazh.github.io/css-landscape/

Задача

Мне стало интересно как далеко можно перенести из HTML в CSS сохраняя максимум поддержки браузерами.

Реализация

Я себе поставил ограничение: один элемент — один тег. Так например у рыбка — это один div, плавник и глаз — это :before, хвост — это :after, чешуя — это градиент. Вся анимация естественно CSS. Парлакс — это правила на a:hover+a+a+a… Таким же образом работают чекбоксы.

Проект выполнен на CSS + HTML.
