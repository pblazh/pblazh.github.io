тест для Datapine

https://pblazh.github.io/datapine-senior-test/

Задача

Заданием было разработать одно-страничный сайт (SPA), как тестовое задание для компании Datapine на должность старшего фронтенд разработчика.

Я должен был создать SPA используя фреймворк Backbone.js, Backbone layout manager и фреймворк для визуализации данных Highcharts. При стилизации страницы необходимо было использовать SASS.

Реализация

Это как раз тот случай, когда лучше посмотреть самому. Я выложил проект на github. Там есть детальное описание задания, мои мысли почему я использовал то или другое и обильно комментированный код. Я думаю, что код сам по себе скажет обо мне больше, чем ещё больше слов тут.
