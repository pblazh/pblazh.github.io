Santa Rap

interactive card

Client: Music TV channel M1

interactive card

http://santa-rap.com/

Task

Come up with the idea of original New Year greetings for viewers the M1 music channel.

Solution

We have created an interactive postcard SantaRap, which enables the user to sing his greetings and send it to friends.

Choose one of two ringer, you can record greetings by a microphone, to create a video series and get an original greeting card. Also, realized the possibility text type with the keyboard — using speech synthesizer text is transformed into song.

The server side is developed in python. A sound is processed using a vocoder and the SOX library. As a speech synthesizer, I used the google translate and ”espeak“ as a backup.

The project is made using HTML5 (Backbone) and Python technology.
