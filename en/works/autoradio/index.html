Autoradio

Client: Autoradio (Ukraine)

http://avtoradio.ua/

Task

The task was to develop an interactive presentation for the biggest commercial radio station in Ukraine. The requirements were quite unusual. The presentation should be in EXE format and its size shouldn't exceed the limit for an email attachment. If the Internet is available the application should automatically renew its content. There also should be a possibility recompiling presentation by a manager with new data.

Solution

I developed the next solution. All data saved in the folders hierarchy in text, CSV and png files. The building program written in python analyze the hierarchy and according to it generates AS code. Later it compiles the project embedding texts and images. Then it transforms SWF into EXE and archives it to send as an attachment.

The project made using Adobe Flash technology.
