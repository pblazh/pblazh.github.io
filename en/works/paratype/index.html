ParaType

Client: ParaType, Agency: Redkeds

/static/works/paratype/index.html

Task

The task was to develop a frontend part for a web platform for selling fonts. The design was created by Redkeds agency.

Solution

As a base, I chose the vanilla Backbone library as I need fine-grained control over the animation and data flow. Most of the animation performed using CSS transitions and more complex ones with the TweenMax JavaScript library.

Sadly when the front end was 90% ready, the agency discovered what except the front part the backend has to be also rewritten and the project was stopped.

The project is made using HTML5 technology (Backbone, TweenMax, Nunjucks).
