Lipton Ice Tea

Client: PepsiCo, Агентство: Tribal DDB

http://dev.liptonicetea.com.ua/

Task

Based on the creative idea and design layout by Tribal DDB advertising agency, to create a website that would allow a great number of various activities that would attract new customers and promote the brand engagement.

Solution

As a base, I chose a Backbone/Marionette framework. The decision was made due to the next factors. Each activity was made as a separate page, the user data should be synchronized between server and client side models and the site has already had a jQuery as a dependency.

Sadly a lot of time was spent on trying to force to make the design WEB compatible and not on developing itself. As a result, the site still has a few drawbacks which I have no possibility to fix.

The project is made using HTML5 technology (Backbone/Marionette).
